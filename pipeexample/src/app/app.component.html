percent Pipe Example

{{ num | percent:'5.4-6' }}

NOte: 5 means 5 digit on left hand side of decimal and 4-6 means minimum 4 digit after decimal and maximum 6 digits ...output will be 01,000.0000%

date Pipe Example

birthday is {{ birthday | date }}

Below date pipe has given format so it will convert date in given format

birthday is {{ birthday | date:"MM/dd/yy" }}

date Pipe Example

Below date pipe is used with uppercase pipe so we can use multiple pipes for formatting text

{{ birthday | date:'fullDate' | uppercase}}
